2-БОБ — LIQUIDITY ENGINE
2.1 Liquidity нима?
Бозорнинг ҳақиқий ёқилғиси — ликвидлик.
Институциялар катта ҳажмда савдо қилиш учун қарши томонда етарлича ордерлар бўлишига муҳтож. Шу сабабли нарх кўпинча аввал стоплар тўпланган жойга боради, кейин асосий ҳаракатни бошлайди.
2.2 Liquidity турлари

Одатда қуйидаги жойларда йиғилади:

Old High 
Double Top 
Range High 
Бу ерда кўплаб Buy Stop ва Short Stop Loss ордерлари бўлади.

Одатда қуйидаги жойларда:

Old Low

Бу ерда Sell Stop ва Long Stop Loss ордерлари жамланади.

2.5 Liquidity Sweep
Liquidity Sweep — нарх ликвидликни олиб, кейин тезда орқага қайтиши.
BUY мисоли:

Кейинги шамлар пастга қайтиб кетади. 
SELL мисоли:

Янги паст ҳосил бўлади. 
Кейин нарх юқорига қайтади.

3-БОБ — ORDER BLOCK ENGINE
3.1 Order Block нима?
Order Block (OB) — бу институционал иштирокчилар катта ҳажмда позиция тўплаган зона деб қаралади. Smart Money ёндашувида нарх кейинчалик шу зоналарга қайтиб реакция бериши мумкин.
Бот Order Block'ни алоҳида сигнал сифатида эмас, балки бошқа тасдиқлар (масалан, тренд, бозор тузилиши ва ликвидлик) билан биргаликда баҳолайди.
3.2 Bullish Order Block
Bullish сценарий учун одатий кетма-кетлик:
Пастга ҳаракат. 
Кучли юқорига импульс. 
Импульс олдиданги охирги bearish шам Order Block сифатида белгиланади. 
Кейинчалик нарх шу зонага қайтса ва бошқа шартлар ҳам мос келса, BUY имконияти баҳоланади. 
3.3 Bearish Order Block
Bearish сценарий учун:
Юқорига ҳаракат. 
Кучли пастга импульс. 
Импульс олдиданги охирги bullish шам Order Block сифатида белгиланади. 
Қайтишдан кейин бошқа тасдиқлар билан бирга SELL баҳоланади. 
3.4 Order Block Quality
Ҳар бир OB бир хил эмас.
Бот қуйидагиларни баҳолайди:
Импульс кучи. 
Order Block ҳажми. 
BOS билан боғлиқлиги. 
Liquidity Sweep мавжудлиги. 
Қайта тестлар сони. 
Волатиллик (ATR). 
Савдо сессияси. 
3.5 Fresh Order Block

ҳали биринчи марта тест қилинаётган; 
бир неча марта бузилмаган; 
кучли импульс билан шаклланган. 
Бундай зоналарга юқорироқ балл берилади.
3.6 Weak Order Block
Қуйидаги ҳолатларда балл пасаяди:

Импульс кучсиз.

Volume паст. 
3.7 Order Block Retest
Бот фақат зонага қайтишни эмас, балки реакцияни ҳам текширади.
Масалан:
Нарх Order Block'га қайтди. 
Рад этиш (rejection) шами ҳосил бўлди. 
Кейинги шам йўналишни тасдиқлади. 
Шундан кейингина сигнал кўриб чиқилади.
3.8 Order Block Rejection
Қабул қилинадиган белгилари:
Узун соя (wick). 
Кучли танали шам. 
Кетма-кет бир нечта тасдиқловчи шам. 
Бу белгилари йўқ бўлса, бот сигнални кучсиз деб баҳолайди.

4-БОБ — FAIR VALUE GAP (FVG) ENGINE
4.1 Fair Value Gap (FVG) нима?
Fair Value Gap (FVG) — бу учта кетма-кет шам орасида нарх тўлиқ савдо қилмаган (номутаносиб) ҳудуд сифатида қаралади. Smart Money ёндашувида нарх баъзан шу ҳудудга қайтиб, кейин асосий ҳаракатини давом эттириши мумкин.
Бот FVG'ни мустақил сигнал сифатида эмас, балки бошқа тасдиқлар билан бирга баҳолайди.
4.2 Bullish FVG
Оддий сценарий:
Кучли юқорига импульс.
Учта шам орасида пастки ва юқори чегара орасида бўшлиқ ҳосил бўлади.
Кейинчалик нарх шу ҳудудга қайтса, бошқа шартлар ҳам мос келса, BUY имконияти баҳоланади.
4.3 Bearish FVG
Оддий сценарий:
Кучли пастга импульс.
Учта шам орасида FVG ҳосил бўлади.
Қайтишдан кейин бошқа тасдиқлар билан SELL имконияти баҳоланади.
4.4 FVG сифати
Бот қуйидагиларни текширади:
Импульс кучи.
FVG ҳажми.
BOS билан боғлиқлиги.
Liquidity Sweep мавжудлиги.
Order Block билан яқинлиги.
ATR ва Volume.
4.5 Fresh FVG
Юқори сифатли FVG:
Янги шаклланган.
Ҳали тўлдирилмаган.
Кучли импульс натижасида ҳосил бўлган.
Бундай зоналарга юқори балл берилади.
4.6 Filled FVG
Агар нарх FVG'ни тўлиқ тўлдирган бўлса, унинг аҳамияти пасайиши мумкин.
Шунинг учун бот бундай зоналарга пастроқ балл беради.
4.7 Inversion FVG
Баъзан аввалги Bullish FVG кейинчалик қаршилик (Resistance) сифатида ишлаши ёки Bearish FVG қўллаб-қувватлаш (Support) вазифасини бажариши мумкин.
Бот бу ҳолатни алоҳида белгилайди.
4.8 Balanced Price Range (BPR)
Агар Bullish ва Bearish FVG бир-бири билан кесишса, Balanced Price Range (BPR) ҳосил бўлиши мумкин.
Бу ҳудудларда бот қўшимча эҳтиёткорлик билан қарор қабул қилади.
